# ProductHexagonal

Практична робота №2. Проєкт за архітектурою "Порти та адаптери" (Hexagonal Architecture).

## Структура
- Domain – сутності + інтерфейси
- Application – сервіси
- Infrastructure – InMemory репозиторій + зовнішній API proxy
- Presentation – консольний застосунок

## Функціонал
- Додавання продукту
- Вивід всіх продуктів
- Пошук продукту за назвою
- Підключення зовнішнього API
